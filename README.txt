################################
How to use this app:
################################

You will need a device running the Ubuntu OS.

Simply copy the whole file (pptx-viewer-v1.0) to the home directory of ubuntu.

Then open the terminal and navigate to the pptx-viewer-v1.0 folder and enter the command: 'python3 app.py'

This will launch the flask app and if there is a PowerPoint in the uploads folder that as well.

Finally, start uploading PowerPoints to your hearts content!
